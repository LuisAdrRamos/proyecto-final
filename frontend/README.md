# Frontend - Proyecto Inventario Distribuido

Este frontend está desarrollado en React + Vite con TailwindCSS, Zustand, React Query y autenticación protegida por token.

## 🚀 Tecnologías usadas

- React 18 + Vite
- TypeScript
- TailwindCSS
- Zustand (gestión de estado)
- React Query
- Axios

## 🔧 Instalación

```bash
npm install
